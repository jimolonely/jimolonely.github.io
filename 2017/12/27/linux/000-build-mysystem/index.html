最新部分

从零开始配置ubuntu

基础配置

  1. 更换下载源： 软件更新里选择从其他站点下载，选择阿里云
  2. 拷贝我的密码脚本到系统（python3 -m http:server）
  3. 打开firefox,登录，firefox会同步插件
  4. 登录ubuntu账户
  5. 打开软件更新，检查更新，更新完毕重启系统（搜索update-manager）
  6. 去搜狗输入法官方网站下载ubuntu的包，安装
  7. 安装网络监控插件：
    1. 安装chrome-gnome-shell： apt install chrome-gnome-shell
    2. 然后在firefox里安装gnome shell integration插件
    3. 访问：https://extensions.gnome.org/extension/120/system-monitor/，安装
  8. 重启系统，使插件和输入法生效
  9. 安装坚果云：去坚果云官网下载安装包，安装
  10. 安装截图软件flame
  11. 配置快捷键：
    1. 命令行界面的： 粘贴改为Ctrl+V
    2. 系统快捷键：
      1. 主目录： 改为WIN+E
      2. 自定义截图快捷键：运行flameshot gui，Ctrl+Alt+A

  12. 登录todoist账号（系统同步），同步清单

    1. 使用ubuntu自带的TODO软件，选择扩展，打开Todoist，就会自动同步Todoist的列表了

  13. 建立常用目录：

  14. ubuntu下使用NVIDIA显卡
